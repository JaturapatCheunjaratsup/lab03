Bootstrap : การใช้งาน Form

Post by Jaturapat Cheunjaratsup, Feb 1, 2017.
Form

การตั้งค่าเบื้องต้นของ Form

    จะมีรูปแบบการควบคุมโดยอัตโนมัติ โดยรูปแบบต้นฉบับเดิมทั้งหมดมี <input>, <textarea> และ <select> elements ด้วย class .form การควบคุมมีความกว้าง 100%


การจัดรูปแบบ Bootstrap มี 3 ประเภท

1.รูปแบบแนวตั้ง (นี้เป็นค่าเริ่มต้น)

2.รูปแบบแนวนอน

3.รูปแบบอินไลน์


ข้อบังคับมาตรฐานสำหรับทั้ง 3 รูปแบบนี้

-ทุกๆ แท็ก html ของฟอร์มที่เราต้องการตรวจสอบค่า จะกำหนด คลาส ชื่อ "form-control css-require" เช่น
<input class="form-control css-require" name="name" type="text" id="name" />

-มีรูปแบบการควบคุม <div class="form-group"> (จำเป็นสำหรับระยะห่างที่เหมาะสม)

-เพิ่ม class .form-control ทั้งหมด <input> , <textarea> และ<select>ลงใน elements


รูปแบบ Bootstrap Form (ค่าเริ่มต้น)


รูปแบบ Bootstrap Inline Form


-เพิ่ม class .form-inline ใน <form> ลง element


รูปแบบ Bootstrap Horizontal Form


-เพิ่ม class .form-horizontal ใน <form> ลง element


-เพิ่ม class .control-label ทั้งหมดใน <label> ลง elements
